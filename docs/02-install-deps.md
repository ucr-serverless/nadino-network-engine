# Installing Dependencies

1. Run `001-env_setup_master.sh` to install softwares.
3. Run `002-env_setup_master.sh` to install libbpf, DPDK, and other build dependencies.

```bash
bash sigcomm-experiment/env-setup/001-env_setup_master.sh

bash sigcomm-experiment/env-setup/002-env_setup_master.sh
```
